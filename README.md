### AzTube Extension

Setup:

1. Clone the repo `git clone https://github.com/NouCake/aztube-extension`
2. Open browser e.g. Chrome
3. Go to Settings > Extensions > Turn on Developer mode
4. Select the folder of the repo you just cloned
